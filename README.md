# Tangle

Tangle is a modern social media platform inspired by Threads and Twitter, designed to connect people through concise and engaging conversations.
![Screenshot 2025-04-11 193444](https://github.com/user-attachments/assets/0e86fc19-4cce-4c88-a462-4e93fce8e02f)


## ✨ Overview

Tangle provides a seamless social networking experience where users can share thoughts, follow interesting people, and engage in meaningful conversations. With a clean and intuitive interface, Tangle makes staying connected more straightforward.

## 🚀 Features

### Authentication
- **Secure OAuth** - Easy sign-in with Google and other OAuth providers
- **Persistent Sessions** - Stay logged in across visits
- **Protected Routes** - Secure content access for authenticated users only

### Social Experience
- **Create Posts** - Share your thoughts with text and media
- **Thread Replies** - Engage in conversations through threaded replies
- **User Profiles** - Customizable profiles with bio and profile pictures

### User Interface
- **Responsive Design** - Perfect experience on any device (mobile, tablet, desktop)
- **Infinite Scroll** - Seamlessly browse through content without pagination
- **Real-time Updates** - See new posts and interactions instantly

### Content Engagement
- **Like** - Interact with content you enjoy
- **Share** - Easily share posts with others
- **Bookmarks** - Save posts to revisit later
- **Search Functionality** - Find users and topics of interest

## 💻 Tech Stack

- Next.js
- React
- Tailwind CSS
- MongoDB
- NextAuth.js

## 📱 Responsive & User-Friendly

Tangle is built with a mobile-first approach, ensuring a smooth and intuitive experience across all devices. The clean UI design and thoughtful UX make navigating and using the platform effortless for users of all tech-savviness levels.

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/Dakshitj14/Tangle.git

# Navigate to the project directory
cd Tangle

# Install dependencies
npm install

# Set up environment variables
# Create a .env.local file based on .env.example

# Run the development server
npm run dev
```
